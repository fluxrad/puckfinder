Puckfinder
==========

Find drop-in hockey in your area

## Install

```
go get github.com/fluxrad/puckfinder
```

## Develop

* In Docker (recommended)

```
docker-compose build
docker-compose up
```

* Running Go natively

```
go run ./main.go
```

## Build

```
docker build
```
